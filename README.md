# Velário — Recruitment Landing Page

A static recruitment landing page for **Projeto Portal — Velário**, a Brazilian tabletop RPG platform with an original authorial universe. The page presents the world lore, business model, content strategy, roadmap, and open positions for founding team members.

## Live Demo

> https://recrutamento-velario.vercel.app

## About the Project

Velário is a Portuguese-language RPG hub built around a rich original universe divided into three kingdoms — Veliganur (Adventure), Pacto Adamantino (War/Steampunk), and Teocracia de Astéria (Holy). The platform targets the fragmented Brazilian online RPG market with a freemium model, monthly official adventures, a community journal, and Twitch-based content programming.

This page serves as a public-facing recruitment tool to attract founding contributors ("Arquitetos") across five open areas:

- Media & Social Networks
- Lore & Creative Writing
- Design & Visual Identity
- Scriptwriting & Research
- RPG Systems & Mechanics

## Tech Stack

- Pure HTML5, CSS3, and vanilla JavaScript — zero dependencies
- Google Fonts: Cinzel, Crimson Pro, Space Mono
- Fully responsive (mobile breakpoint at 700px)
- Animated particle system and scroll-triggered nav

## Deploying to Vercel

This is a static site — no build step required.

1. Import this repository on [vercel.com](https://vercel.com)
2. Vercel auto-detects it as a static project
3. Click **Deploy** — done

## Running Locally

No server needed. Just open `index.html` in any browser:

```bash
# Option 1 — open directly
start index.html

# Option 2 — serve with VS Code Live Server or any static server
npx serve .
```
